Environmental Factors in Indoor Navigation Based on Real-World Trajectories of Blind Users


Dataset Description Download

  • Trajectories of blind users in real-world settings.
  • Contextual information, including floorplan map, obstacles and surrounding objects, and instructions type.
  • Estimated and annotated ground truth position in order to test scalability of approaches.
  • Publicly available dataset and code.

Trajectory Data Description

The dataset was collected in a multi-floor mall, with blind users following app-provided turn-by-turn navigation instructions. There were three routes in total. We provide the timestamped collected estimated and annotated trajectories, as well as associated floorplans, landmarks, and functions to compute the evnironmental factors described in the CHI paper. The provided code also has data visualization scripts. Below are the trajectories of the 9 partcipants in the basement floor for the first and third routes.
